# diacare-frontend-auth-ui
# 💻 DiaCare Frontend - Authentication UI

## 📌 Project Overview

This repository contains the frontend implementation of the **DiaCare platform authentication system**, built with Angular.

It provides a modern UI for user registration, login, and account activation.

---

## 🚀 Features

### 🔐 Authentication UI

* Signup & Login forms
* Multi-role support:

  * Admin
  * Patient
  * Doctor
  * Nutritionist

---

### 📧 Account Activation

* Displays activation status after email verification
* Handles success & error messages from backend

---

### 👨‍⚕️ Doctor & Nutritionist Registration

* Upload certificate (PDF/Image)
* Form validation
* Waiting state for admin approval

---

### 🧾 Forms & Validation

* Reactive Forms
* Input validation (email, password, required fields)
* Error handling & user feedback

---

### 🔗 Backend Integration

* REST API communication with Spring Boot backend
* Handles:

  * Multipart file upload
  * Authentication requests
  * Activation responses

---

## 🛠️ Technologies Used

* Angular
* TypeScript
* Reactive Forms
* HttpClient
* HTML / CSS

---

## 📂 Structure

* Feature-based structure
* Auth module:

  * doctor-auth
  * login/signup components
